# Event Management System

## Project Description
This is a web-based Event Management System that allows users to browse events, register for an account, and sign up for events they're interested in attending. The system features user authentication, event browsing, and personal dashboards for users to manage their registered events.

## Technologies Used
- Backend: Node.js, Express.js
- Frontend: HTML, CSS, JavaScript
- Styling: Bootstrap 5
- Data Storage: JSON files
- Authentication: Express-session, cookies

## Features
1. User Authentication (registration, login, logout)
2. Event Browsing and Registration
3. User Dashboard
4. Session Management with Cookies
5. Server-side Data Storage using JSON

## Project Requirements Fulfilled
1. Used Node.js and JavaScript for server-side and client-side scripting
2. Implemented HTML and CSS with Bootstrap for UI
3. Served static data (HTML, images, CSS, JS files) from the server
4. Used fetch function for HTTP requests to the server
5. Implemented a registration form with POST method
6. Saved form data on the server using JSON files
7. Implemented cookies and session management (bonus)

## Setup Instructions
1. Install dependencies: npm install
2. Initialize data: node init-data.js
3. Generate placeholder images: node create-images.js
4. Start the server: node server.js
5. Access the application at http://localhost:3000
